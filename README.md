## 🚀 Key Project: `BLUSK` Language
**BLUSK**는 생산성과 제어력을 동시에 극대화하기 위해 개발 중인 독자적인 프로그래밍 언어입니다.

- **#LLVM Native Compilation:** LLVM 백엔드를 활용한 강력한 네이티브 기계어 최적화 컴파일 지원
- **Custom Compiler & VM:** 자체 제작한 프론트엔드/백엔드 컴파일러 파이프라인 및 아키텍처 독립적 VM 운영
- **High/Low-Level Productivity:** 높은 추상화 수준의 부드러운 구문과 하드웨어 제어 능력을 동시에 제공
- **Explicit Resource & Memory Management:** 안전하고 직관적인 메모리 관리 메커니즘과 명시적인 리소스 수명 제어
- (추후 1.5버전부터 부트스트랩(셀프코드 바이너리 컴파일) 예정)
